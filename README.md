# BHOJFLOW

Smart QR Ordering & Restaurant Management Platform.

## Monorepo Structure

- `apps/customer`: Customer-facing ordering app (Vite + React)
- `apps/dashboard`: Restaurant management dashboard (Vite + React)
- `apps/kds`: Kitchen Display System (Vite + React)
- `packages/api`: Backend API (Express + Socket.io + Prisma)
- `infrastructure/prisma`: Database schema and client
- `packages/shared`: Shared types and logic
